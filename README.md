# conference_planner_thesis
Conferences planner for institutions

# Run the project
You must have `yarn`:
`npm i -g yarn`

If you are on a mac only run: 

* `npm star`

In other OS:

* Client side
	* `cd client`
	* `yarn`
	* `yarn start`

* Server side (from root)(must have mongod running)
	* `cd server`
	* `yarn`
	* `npm run dev`
